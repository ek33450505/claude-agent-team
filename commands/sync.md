Run `cast-sync.sh` to sync agent definitions, config, commands, and skills from the claude-agent-team repository into `~/.claude/`.

$ARGUMENTS

Runs: `bash ~/Projects/personal/claude-agent-team/scripts/cast-sync.sh`

Use after making changes to agents, routing-table.json, agent-groups.json, or commands in the repo to make them active in the current Claude Code session.
